- ההתחברות מול Yemot API (`Login` → token, ~45 דק'). לשמור token ב-session בלבד, לא סיסמה בקוד.
- הסיסמה: יוסף ציין שקשורה לטלפון של עמנואל (`0548451402`) — **לברר/לאמת מול יוסף**.

## פעולות (v1)
1. **העלאת הקלטה לשלוחה 1** — העלאת קובץ אודיו (או הקלטה מהדפדפן) → `UploadFile`/`UploadTextFile` ל-`ivr2:/1`.
2. **הפעלת צינתוק חינמי** — לנרשמים לצינתוק החינמי (ראה [[yemot-tzintuk-guide]]: המנהל לא יכול להוסיף מספרים לבד — רק מי שנרשם בעצמו; RunTzintuk לא-חינמי נחסם ב-0 יתרה, אז להשתמש במנגנון הצינתוק-החינמי הקיים).
- הרחבות עתידיות: ניהול שלוחות, האזנה להקלטות, סטטיסטיקות.

## אתגר טכני — CORS
Yemot API (`call2all.co.il/ym/api`) כנראה **לא מאפשר קריאות דפדפן ישירות (CORS)**. פתרון: **Supabase Edge Function** כפרוקסי (האפליקציה כבר על Supabase) — הדפדפן קורא ל-Edge Function, היא קוראת ל-Yemot server-side ומחזירה. גם מסתיר את ה-token מהרשת של הלקוח.
